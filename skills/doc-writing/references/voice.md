# 本人の声（語彙・記法・反面リスト）

2026-07 時点の観測。新しい本人手書き文書と突き合わせて更新する。

## 語彙（本人が使う言葉をそのまま使う。類義語に言い換えない）

- 仕組み・運用系: 棚卸 / 留置 / 昇格 / 在庫・隠し在庫 / タネとして温存 / 装置 / 道具・道具立て / 正本 / 型・基本の型 / 定点観測 / 見える化 / すり合わせ
- 進め方系: 叩き台（たたき台）/ 壁打ち / 荷下ろし / 座組 / ボール・ボール回し / 砕く / お尻（期日）/ 磨き込み / 発散・収束 / バッチサイズ / ソフトランディング / 良い所どり / 小分けに / 素振り
- 概念系: 視座 / 板挟み / 空中戦・地に足のついた / 泥臭い・地道 / 一等星 / フロー効率・リソース効率 / 偶有的複雑性 / バス係数 / 手触り・手応え / 資本（信頼という名の）/ 感情ケア / コトに向かう / 「正解」にしていく / 透明性・検査・適応
- 禁止語: 「カノニカル」（代替: 正本・唯一の真実・公式・基準）

## 語尾・確度のグラデーション

- 確信・骨格: 「〜とする」「〜こと。」「〜が不可欠です」「〜を提案します」「〜と信じています」
- 仮説・打診: 「〜かも」「〜な気がする」「〜そう」「〜あっても良さそうだなぁと感じています」「〜のでは説」「結論めいた考え方」
- レポートの考察: 事実は「〜という結果になった」、主観は「〜と推測する」「〜の現れのように感じた」「〜と言えそうである」
- 私見の自問形: 「〜のではないか？と私は思っています」
- 口語の言い切り終止: 「revert すればいいだけなので。」
- 伝聞・自信の程度を保存: 「〜しているようなので」「〜という話があったので」「〜という判断から」
- 一人称は「わたし」「私」混在、自称「よなお」。依頼は「〜してほしいです」と直截（過剰敬語なし）

## 記法

- ※ 注記の多用（例外・条件・背景・質問の意図）。強いものは「※注意※」
- 太字は句単位でピンポイント（文単位で塗らない）
- 色: ピンク＝最重要・注意、赤＝今回の変更点
- 取り消し線＝経緯の可視化（消した内容を残す。旧称もタイトルに残す）
- 💡 コールアウト＝定義・サマリー・ルールの背景
- 絵文字は機能的に: ページアイコン / 分類プリフィックス（🎨🔧📊、凡例付き）/ 感情の緩衝材（お願いの文末🙏）。見出し装飾・文末の😊✨連打はしない
- ラベル prefix: 「仮説：」「相談：」「（共有のみ）」「（できれば）」「（実験的）」「（本音：…）」— 本音や毒はラベルで隔離する
- ❌よくない例／✅よい例 のセリフ単位対比。☝️で直前項目を指す
- 括弧での即時定義: 「影響力(社内で物事を動かす力)」
- 「FYI:」「etcetc…」。長音の感情表現「助かりましたー！」「嬉しい〜〜〜！」、「笑」「笑笑」
- タイトルは動詞終止が基本（「〜できるようにする」）。まれに遊び（「〜にお別れ」「bye-bye 〜」）

## 対人の配慮（先回り型）

- 防御解除: 「特定の個人の評価を目的とするものではありません」「〜と言いたいのではなく〜という意図です」
- 批判的読者の承認: 「あなたはとても鋭いです」と認めてから種明かし
- 個人の願いとして提示: 「個人的には推奨したいです」「個人的にお願いしたいこと」
- 感謝を冒頭・末尾に高頻度で置く。強い言葉はまれに、しかし鮮烈に使う（普段が柔らかいぶん効く設計）
- 個人批判は公開文書に書かない。批判するのは常に「案」

## 反面リスト（本人が書かないもの = 書いたら本人らしさが壊れる）

1. em-dash 箇条書き「**タイトル** — 説明」（役割語＋コロン or 段下げで構造化する）
2. 「まとめ」「おわりに」型の定型締め（ガイドラインは実務項目でスパッと終わる）
3. 対称的に整った網羅列挙（未決は xxx / TBD / ？ のまま公開する）
4. 空のカタカナ語（シナジー等）。抽象概念は括弧で即時具体化する
5. 三点並列の美文（修辞のための列挙）
6. 滑らかすぎる文法（完璧に磨かれた文は本人らしくない。誤字を模倣する必要はないが、磨きすぎない）
7. 「〜とは？」型のSEO的見出し（疑問形見出しは Why 系のみ）
8. 両論併記の中立化（各項目で立場を言い切る）
9. 経緯を消す清書（取り消し線で経緯を残す文化）
10. 英語セクションラベル（Why / What / Summary でなく「なぜやるのか」「設計判断」「制約条件」）
11. 「〜ためです」の機械的反復（理由表現は「〜ので」「〜という判断から」「（〜ため）」と揺らぐ）
12. 網羅的な変更ファイル一覧（変更は意味単位 3〜6 bullet）
13. 数値なしの成果主張（必ず数字。伝聞は伝聞と明記）
14. 命令形・締め切り強要（「しませんか」「いただけると嬉しい」「頼みます＋頼りにしてます」）
15. 中身のない定型句・空回し導入（一文目から具体に入る）
16. 説得段落・ストーリー営業文（「まさに」「これこそが」「〜として誰にでも読める」）
17. 見出しの装飾タイムボックス「（15分）」等、本人の同種文書に無いメタ情報
